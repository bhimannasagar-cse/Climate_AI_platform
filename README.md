## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/bhimannasagar-cse/Climate_AI_platform.git
```

Move into the project folder:

```bash
cd Climate_AI_platform
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
